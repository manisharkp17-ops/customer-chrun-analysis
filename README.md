# Customer Churn Analysis Using Python

## Project Overview

This project analyzes customer data to understand the factors influencing customer churn. Customer churn occurs when customers stop using a company's services. By examining customer information such as age, balance, credit score, salary, and activity status, the project helps identify patterns related to customer retention and customer loss.

## Objectives

* Analyze customer demographic and financial data.
* Identify factors that influence customer churn.
* Visualize customer behavior using charts and graphs.
* Perform exploratory data analysis (EDA).
* Prepare data for future machine learning models.

## Dataset Features

* Customer ID
* Age
* Gender
* Tenure
* Account Balance
* Credit Score
* Estimated Salary
* Number of Products
* Active Member Status
* Churn Status

## Analysis Performed

* Data inspection and preprocessing
* Missing value analysis
* Statistical summary generation
* Churn distribution analysis
* Age distribution analysis
* Balance versus churn analysis
* Correlation analysis using heatmaps

## Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Results

The analysis provides insights into customer behavior and helps understand how different customer attributes relate to churn. Visualizations make it easier to identify trends and patterns within the dataset.

## Conclusion

Customer churn analysis is important for improving customer retention and business growth. This project demonstrates how data analysis techniques can be used to explore customer behavior and support data-driven decision making.
